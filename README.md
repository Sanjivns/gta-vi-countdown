# GTA VI Countdown ⏳🎮

A premium, immersive countdown webpage for the highly anticipated Grand Theft Auto VI release.

## Features

- **Dynamic Countdown**: Live, real-time countdown to November 19, 2026.
- **4K Background Slideshow**: Beautifully curated background images that cycle with smooth fade transitions.
- **Premium Design**: Modern glassmorphism UI with Google Sans typography and subtle micro-animations.
- **Mobile Optimized**: Smooth, flicker-free experience on mobile devices using dynamic viewport units (`dvh`).
- **SEO & Performance**: Optimized meta tags for social sharing and image preloading for seamless transitions.
- **"Out Now" State**: Automatically transitions to a sleek "Launched" display with official links once the countdown ends.

## Usage

Simply open `index.html` in any modern web browser to view the countdown.

## Tech Stack

- **Frontend**: Pure HTML5, CSS3 (Vanilla), and ES6+ JavaScript.
- **Typography**: [Google Sans](https://fonts.google.com/specimen/Product+Sans) (via Google Fonts).
- **Optimization**: Image preloading and GPU-accelerated CSS animations.
